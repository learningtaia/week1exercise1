Web Development

Web development is the act of creating and maintaining websites and other web-based tools. It involves the use of many technologies to build and deliver web pages and applications to users. The scope of use is wide, allowing for a highly flexible, dynamic space for innovation and long term sustainability.

Web Technologies

Technology Description
HTML The skeleton of a webpage. It defines whats on the page - headings, paragraphs, images, links, buttons.
CSS Controls the appearance of the webpage. It handles fonts, colours, spacing, layout, rsponsivenes.
JS Adds interactivity and logic. It makes things happen - button clicks, animations, form validation, dynamic content.

Development Tools

Technology Description
IDE The workspace where you write and manage source code for the webpage. Where the website is built.
Git A control version system. It tracks changes to the code and lets you manage projects overtime. A history tracker for the websites source code.
Web Browser Displays the website to users by reading the source code (HTML, CSS, JS) and turning it into a visible, interactive page. Where the website comes to life.
